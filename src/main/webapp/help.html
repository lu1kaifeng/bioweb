I Quick start

1. The first step

(1) The users can choose system from 'System' options.
(2) 'Cancer type' provides cancer type options.
(3) The users can click the 'Go' button.
2. The second step

(1) The users can enter a gene symbol in ‘Gene symbol’text box. The inputs only have the option of using official gene symbol.
(2) 'Data source' provides dataset options. The users can choose to evaluate the gene in an individual cohort or a combined cohort according to their needs.
(3) The users can click the 'Kaplan-Meier plot' button.
(4) Base on the expression of input gene, some candidate thresholds are set for this parameter. After then, by clicking the ‘Kaplan-Meier plot’ button, The server will take the request and return the analysis results, which are graphically displayed and presented with HR, 95% CI and log-rank p value.

II Advanced application

1. The first step

(1) The users can choose system from 'System' options.
(2) 'Cancer type' provides cancer type options.
(3) The users can click the 'Go' button.
2. The second step

(1) The users can enter a gene symbol in ‘Gene symbol’text box. The inputs only have the option of using official gene symbol.
(2) 'Data source' provides dataset options. The users can choose to evaluate the gene in an individual cohort or a combined cohort according to their needs.
(3) Under 'Survival' option, 13 distinct types of survival terms are provided.
(4) By 'Spilt patients' dialog, user could select different thresholds of gene expression levels to divide patients into two, three or four subgroups of input gene.
(5) In addition, some clinical factors including TNM, gender, fibrosis ishak score, histological type and history of hepatocarcinoma risk factors were set as optional factors to filter patients for special purpose from different clinicians and researchers.
(6) The users can click the 'Kaplan-Meier plot' button.
(7) Base on the expression of input gene, some candidate thresholds are set for this parameter. After then, by clicking the ‘Kaplan-Meier plot’ button, The server will take the request and return the analysis results, which are graphically displayed and presented with HR, 95% CI and log-rank p value.
